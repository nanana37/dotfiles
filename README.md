# dotfiles

My dotfiles.

- [x] neovim (with [lazy.nvim](https://github.com/folke/lazy.nvim))
- [x] zsh
- [x] tmux
- [ ] bash (relatively slow for git prompt)

## Installation
```
git clone https://github.com/nanana37/dotfiles.git
cd dotfiles
make install
```
